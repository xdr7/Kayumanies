1. DASHBOARD KASIR

File: modules/kasir/dashboard.php

Tampilan 4 Kolom Status:

Verifikasi Pembayaran	Pesanan yang belum bayar	Konfirmasi Bayar → Status jadi "processing"

Sedang Diproses	Dalam pengerjaan	Siap Diambil → Status jadi "ready"
Siap Diambil	Sudah jadi, tinggal ambil	Selesaikan → Status jadi "completed"

Fitur:

    Filter tanggal (bisa lihat riwayat hari lain)

    Statistik real-time

    Klik card pesanan untuk detail

    Form aksi di setiap card

    Auto refresh setiap 60 detik (hanya hari ini)

    Flash message setelah aksi

4. PROSES AKSI

File: modules/kasir/process.php

Action yang diproses:
A. Konfirmasi Pembayaran
text

Input: order_id, payment_method
Proses:
  1. Validasi pesanan belum dibayar
  2. Validasi stok produk mencukupi
  3. Update payment_status = 'paid'
  4. Update order_status = 'processing'
  5. Insert ke tabel payments
  6. Kurangi stok produk
  7. Kirim notifikasi ke user
  8. Catat activity log

B. Update Status
text

Input: order_id, new_status
Proses:
  1. Validasi flow status (tidak bisa mundur)
  2. Validasi pesanan sudah dibayar
  3. Update order_status
  4. Jika cancelled → kembalikan stok
  5. Kirim notifikasi ke user
  6. Catat activity log

Flow Status:
text

pending → processing → ready → completed
    ↓          ↓          ↓
  cancelled  cancelled  cancelled

# 🔒 Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

Jika Anda menemukan kerentanan keamanan pada platform Kayumanies, harap laporkan dengan mengirim email ke pengembang. Kami akan merespons laporan Anda dalam waktu 48 jam.

Mohon **jangan** membuka *issue* publik untuk kerentanan keamanan yang belum ditambal.
